# DartML

DartML is planned as a production-grade machine learning ecosystem for Dart.

## Current Status

The repository is in Phase 0: governance, workspace, and verification
foundation. No tensor, autograd, neural network, DataFrame, classical ML, or
GPU backend implementation exists yet.

## Phase 0 Focus

- Dart pub workspace.
- Package skeletons.
- Project control documents.
- Verification tooling.
- CI setup.
- Native runtime proof of concept.
- Honest compatibility and benchmark reporting.

## Evidence Rule

No test, benchmark, platform, hardware backend, compatibility, parity, or
performance claim is valid unless it is backed by an executed command or a
recorded skipped/not-verified reason.
