# Code Of Conduct

## Standard

Participants are expected to be respectful, constructive, and precise.

## Engineering Conduct

- Correctness comes before feature count.
- Unsupported behavior must be explicit.
- Evidence must be reported honestly.
- Failures should be preserved when they describe correct expected behavior.

## Reporting

A formal reporting contact has not been selected yet. This is tracked as a
Phase 0 governance gap.
