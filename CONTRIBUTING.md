# Contributing

## Current Scope

Only Phase 0 work is active. Do not add tensors, autograd, neural networks,
DataFrames, classical ML algorithms, or GPU backends during Phase 0.

## Required Discipline

- Read `INSTRUCTION.md` before making changes.
- Keep handwritten source, tests, scripts, native files, and configuration
  files at or below 80 physical lines.
- Run applicable verification commands.
- Report pass, fail, skipped, and not-verified results accurately.
- Do not invent benchmarks, compatibility claims, or hardware evidence.

## Change Shape

Use small, independently verifiable deliverables. Complete and verify one
deliverable before starting another.
